Terminal Based Text Editor

A minimalist terminal text editor in C that reads raw keystrokes and keeps a live on-screen buffer. It uses a linked-list core for near-instant inserts and deletes at the cursor. The code is modular with a clear header/API, boundary checks, and robust error handling. This project showcases low-level I/O, data structures, and performance-minded systems programming.
